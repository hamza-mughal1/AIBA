## How to Contribute

AIBA is open source. If you'd like to contribute:

1. Fork the repo — [github.com/hamza-mughal1/AIBA](https://github.com/hamza-mughal1/AIBA)
2. Create a branch for your feature or fix
3. Open a PR with a clear description

For bugs, feature requests, or questions, open an issue on GitHub or reach out on [LinkedIn](https://www.linkedin.com/in/-hamza-mughal/).
